# mikicara

Proyecto basado en DeepFaceLive para intercambio facial en tiempo real.

## Requisitos

- Python 3.10+
- GPU compatible para mejor rendimiento (opcional)

## Ejecucion

```bash
python main.py run DeepFaceLive
```

## Notas

- Este repositorio excluye modelos pesados para facilitar versionado en GitHub.
- Los modelos se descargan o generan localmente segun el flujo de la aplicacion.
